# Landing Page Prompts

完整落地页提示词合集。

Landing Page 是包含多个区块的完整页面设计，通常包含：
- Hero 首屏区域
- 功能特性展示
- 用户评价 / 社会证明
- 定价方案
- FAQ
- 页脚（Footer）
