## 简介

一款专为高端服务（如私人飞机、顶级礼宾、豪车租赁等）设计的极简风落地页首屏。它摒弃了冗余的元素，仅通过高质量的背景循环视频与精巧的排版来传递品牌的高级感。文字排布利用了 `leading-none` 与负边距（`-mt-3`）创造出两行巨大标题的紧密重叠效果，再搭配高冷内敛的灰度配色方案（`#202A36`、`gray-500`），将“奢华”与“可触及”的对比感拉满。

### 亮点特性

- **冷色调沉浸式视频背景**：原生 HTML5 Video 实现的全屏沉浸式背景（配合 `object-cover` 撑满视口），不设任何黑色遮罩，直接与灰色调的前景文字发生化学反应
- **紧凑型重叠排版（Overlapping Typography）**：利用负边距 `negative margin-top: -12px` 和极紧的字母间距 `tracking-tighter`，让“Premium.”与“Accessible.”两行巨大字体重叠，打造时尚杂志般的视觉张力
- **悬浮折叠式导航（Floating Navbar）**：适配移动端的毛玻璃下拉菜单（`white/95` opacity + `backdrop-blur`），并通过状态钩子 `useState` 精确控制显隐
- **高阶克制用色**：未使用任何高饱和度色彩，按钮和文字使用了特定的深蓝灰（`#202A36`）以及浅灰体系，带来强烈的专业与高品质属性

### 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS 3
- Lucide React (图标)
- 原生 React 状态控制 (useState)

## 使用说明

- **推荐 AI 工具**：Cursor / Claude / ChatGPT
- **使用方式**：复制 [prompt.md](prompt.md) 中的提示词，在 AI 工具中生成完整项目代码
- **注意事项**：
  - 代码对 `Inter` 字体有强依赖，生成后请确保 `index.html` 中已正确引入对应权重 (400, 500, 600, 700)
  - 此模板高度依赖背景视频本身的质量。若需替换，推荐使用高明度的无人机航拍或高端交通工具静止长镜头，避免深色视频破坏灰色字体的对比度
