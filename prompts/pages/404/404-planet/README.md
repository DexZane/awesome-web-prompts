## 简介

404 Planet 是一款为云托管服务商 NEXOVA 设计的全屏 404 错误页面。页面背景使用高分辨率的深蓝地球太空视频循环播放，搭配极简响应式导航、液态毛玻璃（Liquid Glass）返回按钮、发光 404 大字以及多列底部页脚导航。

## 核心设计模式

* **全屏视频背景 (Looping Video Background)**：绝对定位且受 `object-cover` 约束的全屏视频，营造极强深邃太空沉浸感。
* **液态毛玻璃按钮 (Liquid Glass Button)**：采用 `-webkit-backdrop-filter` 配合多层渐变边框实现现代玻璃拟态按钮。
* **404 发光文本 (Glow Effect)**：运用多重 `text-shadow` 打造白光漫射的发光效果。
* **交错动效移动菜单 (Stagger Animated Mobile Menu)**：移动端抽屉菜单带有自适应淡入及交错延迟显示效果。

## 技术栈要求

* Tailwind CSS
* Lucide React Icons
