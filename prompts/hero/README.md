# Hero Section Prompts

首屏主视觉区域提示词合集。

Hero Section 是用户打开网页时看到的第一个区块，通常包含：
- 大标题 / 副标题
- CTA 按钮（Call to Action）
- 背景动画 / 渐变 / 图片
- 导航栏
