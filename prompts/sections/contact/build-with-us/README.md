## 简介

一个为创意设计/开发团队 **Forma** 设计的 Contact Landing Page。页面包含全屏沉浸式微电影级别视频背景，外嵌双层精致的悬浮大圆角卡片。右侧表单采用极佳的物理阴影效果，支持完整的客户端输入状态、标签多选及逼真的 1 秒延迟假提交成功动画，是极其理想的团队联系展示页。

### 亮点特性

- **精致多端自适应布局**：采用 `calc(100vh - 48px)` 圆角外框，在桌面端严格锁定视口，在平板/移动端随着表单内容高度自动撑开。
- **高对比排版反差**：利用极简现代的 Inter 字体作为 UI 骨架，在白色标题中使用 `Instrument Serif` 斜体混排高亮字 `products`，体现工作室的设计水准。
- **多功能标签交互**：支持服务项目的多选 Toggle 状态，拥有顺畅的聚焦/悬停颜色过渡类。
- **精美成功状态回显**：表单在提交时支持 Loading 状态（"Sending..."），并在提交成功后渐隐替换为带有绿色打勾徽章、标题和子文案的感谢卡片。

### 技术栈

- React
- Tailwind CSS
- lucide-react (图标：Twitter, Circle, Instagram, Linkedin)
- Google Fonts (Inter + Instrument Serif)

## 使用说明

- **推荐 AI 工具**：Cursor / Claude / ChatGPT
- **使用方式**：复制 [prompt.md](prompt.md) 中的提示词，在 AI 工具中生成完整项目代码
- **注意事项**：
  - 需要在主 CSS 文件中导入指定的 Google Fonts。
  - 导航条使用了 `backdrop-blur-md`，需要配合 `bg-white/60` 使用以获得最佳磨砂透明度效果。
