## 简介

该提示词用于生成一个包含“行动号召（CTA）”和“底部导航页脚（Footer）”的现代网页组件。设计采用 Tailwind CSS，具有暗色背景、霓虹黄高亮、以及平滑的滚动渐显动画（fade-up），特别是底部还有一个与容器完美对齐的超大 SVG 文本“Tenlas”。非常适合具有科技感或现代感的 SaaS 落地页。

## 使用说明

- **推荐 AI 工具**：Claude  / Cursor / v0
- **使用方式**：直接复制 [prompt.md](prompt.md) 中的提示词粘贴即可
- **注意事项**：生成的代码由于包含了 `IntersectionObserver`，需要在支持现代 JavaScript 的浏览器中预览。SVG 文本的排版在部分生成结果中可能需要微调 `viewBox`。
