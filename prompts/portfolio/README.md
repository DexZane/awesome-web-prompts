# Portfolio Prompts

个人作品集提示词合集。

Portfolio 页面主要用于展示个人或团队的过往案例与专业技能，通常包含：
- Hero 个人简介
- 作品集展示（核心）
- 技能/服务列表
- 工作履历
- 联系方式
